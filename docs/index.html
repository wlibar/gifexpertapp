Repositorio de Proyectos de Software

Este repositorio contiene ejemplos completos de proyectos de software con la documentación detallada de su arquitectura. Algunos proyectos, a primera vista dan la impresión de estar repetidos, pero en realidad en cada materia tiene alcances, tecnologías, patrones y enfoques distintos, así como atributos de calidad diferentes. De esta manera, cada proyecto es único y se diferencia de los demás.
